# Back-end — Text Summarizer

Minimal Node.js backend for the Text Summarizer project.

Prerequisites
- Node.js 14+ and npm

Install
- `cd back-end`
- `npm install`

Run (development)
- `npm start` (if provided) or `node server.js`

Configuration
- `PORT` — port to listen on (default: `3000`)
- Add API keys or other secrets via environment variables or a `.env` file (do not commit secrets).

Notes
- The main entry is `server.js` in this folder.
